# Solidity简介

Solidity 是一种面向合约的高级编程语言，主要用于在以太坊等区块链平台上编写智能合约。

- 语法类似于JavaScript、C++等
- 支持继承、库、复杂用户自定义类型
- 广泛应用于DeFi、NFT、DAO等区块链项目

## 主要应用场景

- 数字资产（Token、NFT）
- 去中心化金融（DeFi）
- 去中心化自治组织（DAO）
- 区块链游戏等
