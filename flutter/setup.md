# 环境搭建

## 1. 安装Flutter SDK
- 访问 [Flutter官网](https://flutter.dev/) 下载对应平台的SDK。
- 解压并配置环境变量（如 `PATH`）。

## 2. 安装开发工具
- 推荐使用 VS Code 或 Android Studio。
- 安装 Flutter 和 Dart 插件。

## 3. 验证安装
```bash
flutter doctor
```
确保所有依赖项均已通过检查。

## 4. 配置模拟器/真机
- Android：安装 Android Studio 并配置 AVD。
- iOS：需 macOS 并安装 Xcode。

详细步骤可参考官方文档。
