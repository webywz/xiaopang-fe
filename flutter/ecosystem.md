# 生态与插件

## 插件市场
- 官方插件市场：[pub.dev](https://pub.dev/)

## 常用插件推荐
- `http`：网络请求
- `provider`：状态管理
- `shared_preferences`：本地存储
- `sqflite`：数据库
- `image_picker`：图片选择
- `url_launcher`：打开外部链接
- `dio`：强大网络库

## 插件安装
```bash
flutter pub add 插件名
```

## 插件使用
详见各插件文档和 pub.dev 示例
